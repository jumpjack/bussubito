Carica file di trasporti pubblici in formato GTFS e permette sia di trovare la linea diretta che unisce partenza e arrivo, sia tutte le linee passanti per partenza e arrivo, in modo da scegliere da soli il tragitto migliore.

Demo: https://jumpjack.github.io/bussubito/index.html

Data file: https://dati.comune.roma.it/catalog/dataset/a7dadb4a-66ae-4eff-8ded-a102064702ba/resource/266d82e1-ba53-4510-8a81-370880c4678f/download/rome_static_gtfs.zip
